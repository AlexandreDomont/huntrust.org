Create sitemap.xml and robots.txt for crawlers - SEO

May 18, 2015 by Alex

It is difficult to exist remaining hidden !

I followed the advice, which advocate to help crawlers find my blog.

Here's an example of what I say and that is found in numbers (in addition, explanations of robots and sitemaps files) :

  • advancedhtml.co.uk
  • www.robotstxt.org
  • www.sitemaps.org

In my case, I created a python script that generates the file sitemap.xml.

The script browse the "items" folder and adds the items in the file sitemap.xml

#!/usr/bin/env python

import os.path, sys, glob, re, os

NomFichier = 'blog/sitemap.xml'
Fichier = open(NomFichier,'w')

Fichier.write('')
Fichier.write('\n')
Fichier.write('\n')
Fichier.write('\nhttp://huntrust.org/')
Fichier.write('\n')
Fichier.write('\n')
Fichier.write('\nhttp://huntrust.org/about.html')
Fichier.write('\n')

for file in glob.iglob("/var/www/huntrust/blog/articles/*.html"):
        	article = file[32:]
		Fichier.write('\n')
		Fichier.write('\nhttp://huntrust.org/articles/'+ article + '')
		Fichier.write('\n')

Fichier.write('\n')

Fichier.close()
