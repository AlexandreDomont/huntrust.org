Creat repo Github and Push code with ssh key

May 7, 2015 by Alex

I use those two procedures for creat and add ssh key

  • create-a-repo
  • generating-ssh-keys

Then push still ask username...

Check configuration repo

git remote -v

And for not add username when push, but use key SSH, add this :

git remote set-url origin git@github.com:AlexandreDomont/huntrust.org.git

Result

git remote -v
origin	git@github.com:AlexandreDomont/huntrust.org.git (fetch)
origin	git@github.com:AlexandreDomont/huntrust.org.git (push)

Go ...

git push origin master
